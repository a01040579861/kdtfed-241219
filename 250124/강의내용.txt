SCSS 중요문법

1. @at-root : 최상위 루트를 기준으로 그안에 해당 선택자(class,id) 태그를 찾아오겠다.

2. variable : 굳이 :root 및 var() 불필요!
color: $main-red-color;

3. @import: 굳이 url() 경로 작성 불필요!
직접 경로주소 입력 가능!

4. JS 산술연산자 사용가능: + - * / % 모두 사용가능
VS 기존 css 내 calc() 사용 불필요!

5. @mixin/ @include 스타일 그룹 설정 : 
반복 및 공통으로 사용될 것으로 판단되는 스타일을 하나 정의해놓고, 그때그때마다 가져다놓고 사용가능!
=> 각각의 스타일이 메인 + 공통 스타일을 서브로 적용하는 개념 

6. @extend: 공통스타일이 메인 + 해당 스타일이 필요한 요소들을 불러 모으는 방식

7. @for: 반복문 사용가능 
> @for $i from 1 through 9 : $i 라는 변수에 1부터 9까지의 값을 넣어주겠다.

8. @map-get: 객체 사용가능
