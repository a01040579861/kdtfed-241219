1- 문서 로딩 이벤트 
- 브라우저를 통해서 특정 웹페이지 도착
- 사용자가 하는 행위 모든 것
- 페이지를 확인 // 버튼 클릭 // 스크롤 이동
- 이벤트를 설정
- event handler => onClick
- addEventListener() 
- 이벤트 설정 > 이벤트 실행 > 로컬 컴퓨터 > 메모리 공간 > 이벤트 객체 생성 > 이벤트 객체를 참조변수 값으로 찾아올 수 있다.
- 이벤트 객체 > preventDefault() 유일한 메서드 함수 

2- 마우스(mouse) 이벤트
- click : 버튼 혹은 특정요소들을 클릭했을 때 실행되도록 하는 이벤트
- dbclick : 버튼 혹은 특정요소들을 클릭했을 때 실행되도록 하는 이벤트
- mouseover : 특정요소 위에 마우스를 올렸을 때 (* 버블링 기본적용)
- mouseenter : 특정요소에서 마우스를 떠나게 했을 때 (* 버블링을 적용하지 않는다.)
- mouseleave : 특정요소에서 마우스를 떠나게 했을 때 (* 버블링을 적용하지 않는다.)

3- 키보드 이벤트
- keydown : 사용자가 키를 누르고 있는 동안 발생되는 이벤트
- keyup : 키를 눌렀다가 키에서 손을 뗄 때 발생하는 이벤트

4- 폼 이벤트
- change  
- focus

- submit

5- 위치 이벤트
- pageX : 문서 (코드를 작성한 작업데이터)를 기준으로 X축으로 부터 얼만큼 이동한 곳인지 확인
- pageY : 문서 (코드를 작성한 작업데이터)를 기준으로 Y축으로 부터 얼만큼 이동한 곳인지 확인
- screenX : 현재 화면을 기준으로 X축으로 부터 얼만큼 이동한 곳인지 확인
- screenY : 현재 화면을 기준으로 Y축으로 부터 얼만큼 이동한 곳인지 확인
